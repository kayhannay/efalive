Neue Statistikarten 'Fahrtart' und 'Welches Boot Wohin' [1.6.0]

Mit der Statistikart Fahrtart lassen sich die Fahrten in Abhängigkeit ihrer Fahrtart, z.B. Training, Ausbildung, Regatta, Wanderfahrt oder normale Fahrt auswerten. Die Statistikart Welches Boot Wohin erstellt eine Matrix aus Booten und Zielen, in der erkennbar ist, welches Boot wie oft wohin gerudert ist.
