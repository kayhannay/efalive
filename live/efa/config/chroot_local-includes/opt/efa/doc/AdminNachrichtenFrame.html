Nachrichten an Admin


Beschreibung

In diesem Fenster werden alle Nachrichten angezeigt, die an den Administrator gesandt wurden. Dies können zum einen Nachrichten sein, die ein normaler Nutzer über die Funktion Nachricht an Admin abgeschickt hat, es können aber auch Informationen oder Fehlermeldungen sein, die efa selbst erstellt hat. Der Administrator sollte daher regelmäßig nachschauen, ob Nachrichten an ihn vorliegen.

Um die Nachrichten anzuzeigen, logge Dich im Hauptfenster als Admin ein und wähle die Funktion Nachrichten anzeigen im Admin-Modus Fenster.

nur ungelesene
 Wenn diese Option gewählt ist, werden nur ungelesene Nachrichten angezeigt. (Ungelesene Nachrichten sind solche, bei denen die Option Nachricht als gelesen markieren nicht gesetzt wurde.)
Nachricht als gelesen markieren
 Wenn diese Option für die angezeigte Nachricht aktiviert ist, wird sie künftig nicht mehr bei den ungelesenen Nachrichten mit angezeigt. Nur Administratoren, die das Recht haben, ungelesene Nachrichten als gelesen zu markieren, können diese Option setzen.

Ok
