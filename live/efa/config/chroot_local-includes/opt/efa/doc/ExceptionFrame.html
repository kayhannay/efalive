Unerwarteter Programmfehler (Exception)


Beschreibung

Wenn Du dieses Fenster siehst, dann bist Du soeben auf einen Programmfehler gestoßen und Java hat eine Exception (Ausnahme) ausgelöst. Die aktuelle Aktion wurde dadurch abgebrochen, efa sollte allerdings anschließend weiterarbeiten.

Damit ich diesen Fehler beheben kann, schicke mir bitte eine email an software@nmichael.de und beschreibe kurz, wobei dieser Fehler aufgetreten ist. Außerdem hänge bitte die Datei efa.log aus dem efa-Verzeichnis an die email an. (Diese Datei enthält ein genaues Fehlerprotokoll, anhand dessen ich den Fehler lokalisieren kann.)
