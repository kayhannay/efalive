efa - elektronisches Fahrtenbuch

Download und Installation

0. Schritt: Voraussetzungen

Software: efa läuft unter allen Betriebssystemen, für die eine Java Virtual Machine in Version 1.4.0 oder neuer existiert (unter anderem die Betriebssysteme Linux, MacOS und Windows ../../2000/XP und einige andere). Für optimale Funktionalität sollte mindestens Java Version 5 eingesetzt werden.

Hardware: Als Mindestanforderungen für einen sinnvollen Einsatz von efa sollte ein Computer mit 100 MHz oder mehr, 48 MB RAM, Festplatte, VGA-Grafikkarte mit 800x600 Bildpunkten und 256 Farben, Tastatur, Maus und 14 Zoll Monitor vorhanden sein. Ein etwas schnellerer Computer mit 64 MB RAM wäre empfehlenswert.


1. Schritt: Download von efa

efa kann als Windows-Installationsprogramm, ZIP-Archiv, ZIP-Quelltextarchiv oder Komplettsystem heruntergeladen werden. Bitte wähle eine der drei folgenden Möglichkeiten:

  • efa183.exe (mit Installationsprogramm; für alle Windows-Nutzer empfohlen)
  • efa183.zip (ZIP-Archiv; für alle Nutzer eines anderen Betriebssystems empfohlen)
  • efa183src.zip (ZIP-Quelltextarchiv)
  • efaLive (efa-Komplettsystem für den Einsatz im Bootshaus)

Sollten diese Download-Links nicht mehr funktionieren, so ist bereits eine neuere Version erschienen. Die Links zur aktuellen Version findest Du hier.

Hinweis zum Update von efa:

  • efa Online-Update: Mit dem Online-Update kann efa mit wenigen Mausklicks die neueste Version von efa aus dem Internet herunterladen und installieren. Wenn Du efa bereits im Einsatz hast und auf dem PC über einen Internet-Zugang verfügst, kannst Du sämtliche hier beschriebenen Schritte ignorieren und stattdessen im efa-Hauptfenster aus dem Menü Datei den Punkt efa Online-Update auswählen.
  • manuelles efa-Update: Wenn Du efa auf einem PC ohne Internet-Zugang aktualisieren möchtest, dann lade bitte eine der oben stehenden Dateien herunter. Vor der Installation der neuen Version muß efa beendet werden. Es empfielt sich, sicherheitshalber alle Daten im Ordner daten zu sichern (auch wenn dies normalerweise nicht erforderlich ist). Anschließend kann die neue Version einfach über die alte Version "drüberinstalliert" werden (also mittels des Installationsprogramms in denselben Ordner installiert werden, bzw. das ZIP-Archiv in dasselbe Verzeichnis entpackt werden). Eine Deinstallation der alten Version ist nicht nötig. Die Einstellungen und Daten bleiben dabei erhalten und können von der neuen Version weitergenutzt werden.

2. Schritt: Installationsvorbereitungen

Wenn Du bereits eine ältere Version von efa in Benutzung hast, lies bitte zuerst diese Hinweise durch.


3. Schritt: Installation von efa

Zur Installation von efa gehe nun, je nach gewählten Download, folgendermaßen vor:

  • wenn Du efa183.exe heruntergeladen hast: Starte das Installationsprogramm durch einen Doppelklick auf efa183.exe und folge den Anweisungen.
  • wenn Du efa183.zip heruntergeladen hast: Erstelle ein neues Verzeichnis und entpacke den gesamten Inhalt des ZIP-Archivs in dieses Verzeichnis.

4. Schritt: Prüfen, ob Java installiert ist

efa benötigt ein installiertes Java Laufzeitsystem (JRE) in der Version 1.4.0 oder neuer. Generell sollte immer eine möglichst aktuelle Java-Version eingesetzt werden, um optimale Funktionalität zu gewährleisten. Die volle Funktionalität von efa ist nur mit Java 5.0 oder neuer gewährleistet. Beste Ergebnisse werden mit der jeweils neuesten Java-Version erzielt, derzeit Java 6.0.
Wenn Du das Installationsprogramm ausgeführt hast, wurde am Ende der Installation geprüft, ob Java installiert ist. Um das Testprogramm erneut zu starten, doppelklicke auf javatest.bat im efa-Verzeichnis. Nutzer von Nicht-Windows Betriebssystemen können in einer Shell (Kommandoprompt) den Befehl java -version eingeben.
Wenn Java nicht (oder in einer zu alten Version) installiert ist, fahre mit Schritt 5 fort, ansonsten mit Schritt 6.


5. Schritt: Java installieren

Wenn Java nicht (oder in einer zu alten Version) installiert ist (siehe Schritt 4), gehe folgendermaßen vor:
Achtung: Folge den einzelnen Schritten bitte genau und beachte, daß es teilweise sehr ähnlich lautende Bezeichnungen für verschiedene Produkte gibt!
Hinweis: Leider ändert SUN die Download-Seiten zu Java von Zeit zu Zeit. Solltest Du feststellen, daß meine Anleitung nicht mehr stimmt, da sich die Seiten geändert haben, schreibe mir bitte eine email.

  • Nutzer von Linux, Solaris und Windows XP/Server-2003/2000/Vista:
    • Öffne die Seite http://java.sun.com/javase/downloads/index.jsp in einem Webbrowser.
    • Klicke in der Rubrik Java Platform, Standard Edition bei JDK 6 Update XX (JDK or JRE) auf Download JRE.
    • Wähle aus der Auswahlliste Platform Dein Betriebssystem aus.
    • Selektiere die Option I agree to the Java SE Runtime Environment ... License Agreement.
    • Klicke Continue.
    • Klicke in der Rubrik Available Files auf den unterlegten Link, um Java herunterzuladen (falls mehrere Links angeboten werden, wähle Dein bevorzugtes Download-Format).
    • Speichere den Download auf Deinem Computer.
    • Doppelklicke nach dem Download auf die heruntergeladene Datei, um das Java-Installationsprogramm zu starten. Du benötigst hierzu Administrator-Rechte auf Deinem Computer.
    • Folge den Anweisungen des Installationsprogramms.
  • Nutzer älterer Windows-Betriebssysteme wie Windows 98SE/ME:
    • Öffne die Seite http://java.sun.com/javase/downloads/index_jdk5.jsp in einem Webbrowser.
    • Klicke in der Rubrik Java Runtime Environment (JRE) 5.0 Update XX auf Download.
    • Da Java 5 bereits das Ende des offiziellen Supports erreicht hat, mußt Du vor dem Download von Java 5 ein Registrierungs-Formular ausfüllen, bevor Du zum Download gelangst.
  • Nutzer noch älterer Windows-Betriebssysteme wie Windows 98 (nicht SE)/NT4-SP6a:
    Achtung: Für den optimalen Einsatz benötigt efa Java 5 oder neuer. efa funktioniert zwar weiterhin auch mit Java 1.4.2, jedoch werden einige Funktionen unter Java 1.4.2 nicht vollständig unterstützt.
    • Öffne die Seite java.sun.com/products/archive in einem Webbrowser.
    • Wähle in der Rubrik Java Platform Standard Edition (Java SE) bei J2SDK/J2RE - 1.4 den Punkt 1.4.2_19 aus (bzw. die neueste Version 1.4.2_xx) und klicke dahinter auf Go.
    • Klicke in der Rubrik J2SE v 1.4.2_xx JRE auf Download J2SE JRE.
    • Da Java 1.4.2 bereits das Ende des offiziellen Supports erreicht hat, mußt Du vor dem Download von Java 1.4.2 ein Registrierungs-Formular ausfüllen, bevor Du zum Download gelangst.
  • Nutzer ganz alter Windows-Betriebssysteme wie Windows 95/NT4-SP5:
    Achtung: Für den optimalen Einsatz benötigt efa Java 5 oder neuer. efa funktioniert zwar weiterhin auch mit Java 1.4.0, jedoch werden einige Funktionen unter Java 1.4.0 nicht vollständig unterstützt.
    • Öffne die Seite java.sun.com/products/archive in einem Webbrowser.
    • Wähle in der Rubrik Java Platform Standard Edition (Java SE) bei J2SDK/J2RE - 1.4 den Punkt 1.4.0_04 + documentation aus und klicke dahinter auf Go.
    • Klicke in der Rubrik Download J2SE v 1.4.0_04 in der Spalte JRE in der Zeile Windows (U.S. English / all languages) auf Download.
    • Wähle aus der Auswahlliste Platform den Punkt Windows aus.
    • Selektiere die Option I agree to the Software License Agreement.
    • Klicke Continue.
    • Klicke in der Rubrik Windows (all languages, including English) auf den blau hinterlegten Link (z.B. j2re-1_4_0_04-windows-i586-i.exe oder ähnlich).
    • Speichere den Download auf Deinem Computer.
    • Doppelklicke nach dem Download auf die heruntergeladene Datei, um das Java-Installationsprogramm zu starten. Du benötigst hierzu Administrator-Rechte auf Deinem Computer.
    • Folge den Anweisungen des Installationsprogramms.

6. Schritt: efa starten

Nun kann efa gestartet werden. Gehe dazu, je nach gewählten Download, folgendermaßen vor:

  • wenn Du efa183.exe heruntergeladen hast: Gehe ins Startmenü und wähle ->Programme->efa->efa - elektronisches Fahrtenbuch.
  • wenn Du efa183.zip heruntergeladen hast: Für Windows kann die Datei efa.bat, für Linux die Datei efa.sh als Startskript benutzt werden. Nutzer anderer Betriebssysteme müssen efa analog zu den Aufrufen in diesen beiden Startdateien aufrufen.
    Hinweis: Das Startskript muß aus dem efa-Verzeichnis heraus aufgerufen werden! Unter Unix-ähnlichen Betriebssystemen kann dazu zunächst in der Shell in das efa-Verzeichnis gewechselt werden (cd /mein/efaVerzeichnis) und dann mittels ./efa.sh das Startkript aufgerufen werden.
