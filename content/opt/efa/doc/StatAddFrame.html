Statistikeinstellungen speichern


Beschreibung

efa bietet die Möglichkeit, häufig benötigte Statistikeinstellungen zu speichern, so daß nicht jedes Mal alle Optionen neu eingestellt werden müssen.

Wenn Du die aktuell gewählten Statistikeinstellungen speichern möchtest, so klicke im Statistik Erstellen-Fenster auf den Button Statistikeinstellungen speichern.


Erklï¿½rung der Buttons und Felder
Bezeichnung für aktuelle Statistikeinstellungen
 Gib hier einen Namen für die Einstellungen ein, unter dem sie gespeichert werden soll.
Einstellungen als Standardeinstellungen speichern
 Durch wählen dieser Option werden die aktuellen Einstellungen zu den Standardeinstellungen, d.h. sie werden bei jedem Öffnen des Statistik-Fensters automatisch gesetzt. Wenn diese Option gewählt ist, kann kein Name für die Einstellungen vergeben werden.
Statistik auch im Bootshaus verfügbar machen
 Wenn diese Option für eine Statistik gewählt ist, so kann sie auch aus der Bootshaus-Version von efa von jedem Mitglied abgerufen werden.
Hinweis: Es stehen im Bootshaus nur solche Statistiken zur Verfügung, die zuvor mit dieser Option gespeichert wurden!
Statistikeinstellungen speichern
 Speichert die Einstellungen.
