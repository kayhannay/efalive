Fenstereigenschaften (Bootshaus) [1.4.0]

Damit Mitglieder sich nicht den Spaß machen können, das efa-Fenster bis an den Rand des Bildschirms zu verschieben oder es hinter anderen Fenstern zu verstecken, läßt sich dies jetzt verhindern: Auf Wunsch bleibt das efa-Hauptfenster immer an seiner Position (Hauptfenster nicht verschiebbar) und kann nicht durch andere Fenster verdeckt werden (efa immer im Vordergrund). Diese beiden Optionen lassen sich in den efa-Einstellungen auf der Registerkarte efa im Bootshaus / Erscheinungsbild wählen.

Hinweis: Die Option Hauptfenster nicht verschiebbar wird erst ab Java Version 1.4.x unterstützt. Für die Option efa immer im Vordergrund wird Java ab Version 1.5.0 benötigt. Java v1.5.0 wird jedoch erst im Herbst 2004 erscheinen. Wer diese Funktion benötigt, kann sich aber im Internet Java 1.5.0beta als Vorab-Version herunterladen. Diese Version scheint stabil zu funktionieren. Im Allgemeinen rate ich aber derzeit noch zum Einsatz der stabilen Java-Version 1.4.2.
