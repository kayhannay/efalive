Standardmannschaft

Für jedes Boot kann eine Standardmannschaft eingegeben werden, die bei der Auswahl des Bootes automatisch in die Felder eingetragen wird. Dies ist dann sinnvoll, wenn es bestimmte Trainingsboote gibt, die immer von derselben Mannschaft gerudert werden. Desweiteren lassen sich auch ein Standardziel, sowie eine Standard-Fahrtart (z.B. Training) konfigurieren.

Um für ein Boot eine Standardmannschaft auszuwählen, gehe nun zu einem beliebigen Eintrag im Fahrtenbuch und klicke auf den Button hinter dem Bootsnamen. Das Fenster Boot bearbeiten für dieses Boot öffnet sich. Klicke hier auf Standardmannschaft konfigurieren und fülle wahlweise einige der Felder Steuermann, Mannschaft oder Ziel aus und klicke anschließend auf Eintrag speichern, sowie in dem Boot bearbeiten Fenster auf Eintrag übernehmen.
