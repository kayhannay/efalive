Rudererlaubnis - 2 (efa im Bootshaus) [1.7.0]

Zusätzlich zu den eben geschilderten Gruppen läßt sich für jedes Boot noch eine weitere Gruppe definieren, aus der mindestens ein Ruderer oder Steuermann im Boot stammen muß. Dies kann z.B. verwendet werden, um sicherzustellen, daß in jedem Boot mindestens ein Obmann sitzt: Man erstellt hierzu eine Gruppe Obleute, in die man alle Obleute einträgt. Anschließend trägt man für alle gewünschten Boote die Gruppe Obleute ein. efa prüft dann beim Beginnen jeder Fahrt, ob mindestens eine Person im Boot dieser Gruppe angehört und warnt anderenfalls.
